아이디어 정리


스킬 + 룬을 통해 몰려오는 적을 처치하고 20분동안 살아남자



- 목표 : 20분동안 살아남기

몬스터가 끊임없이 지속적으로 몰려와야 함

레벨업 : 몬스터 처치 시 레벨 증가

레벨업 보상 : 스킬, 룬, 장비(차후)

기본 공격 - 룬 3개 장착

스킬 : 쿨타임마다 장착한 스킬 발동

최대 장착 스킬 : 5개(액티브+패시브)
액티브 스킬 - 룬 3개 장착 가능
패시브 스킬 - 룬 장착 불가

as

[룬 시스템]
- 태그
스킬에 태그 붙여서 해당 태그가 알맞아야 적용


ex) 화염구 : X초마다 거대한 화염구를 무작위 방향으로 발사하여 적에게 공격력의 A%만큼 준다.
tag : 투사체, 화염, 쿨타임, 단일, 화상

-> 투사체 증가 룬 장착 시 : 투사체 개수 증가, 투사체 발사 속도 증가, 투사체 크기 증가, 투사체 분리
-> 화염 관련 룬 장착 시 :  [화염] 태그를 가진 스킬에 장착 시 폭발 효과를 가져 적에게 명중 시 폭발하여 광역 피해를 주고 해당 위치를 X초간 불태웁니다. [폭발] 태그를 가집니다. / 날아가는 동안 피해
-> 쿨타임 관련 룬 장착 시 : 해당 스킬의 쿨타임을 X초 감소시킵니다.
-> 화상 관련 룬 장착 시 : [화상] 태그를 가진 스킬에 장착 시 화상 효과를 부여하여 대상에게 X초 동안 A의 피해를 줍니다.
